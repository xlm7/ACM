# [零起点学算法系列](https://github.com/xlm7/ACM/tree/master/begin)

###CountLetters<br><br>
Description<br>

输入一串字符，统计这串字符里的字母个数，数字个数，空格字数以及其他字符(最多不超过100个字符）

Input

多组测试数据，每行一组

Output

每组输出一行，分别是字母个数，数字个数，空格字数以及其他字符个数


    Sample Input 
    I am a student in class 1.
    I think I can!

Sample Output

    18 1 6 1
    10 0 3 1
